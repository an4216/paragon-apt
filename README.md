# 주안센트럴파라곤 입주 안내 (비공식)

입주 예정자를 위한 통합 안내 페이지. 일정·잔금·대출·이사·자료까지 한 곳에서.

## 단지 정보 (검증된 사실)

- **위치**: 인천광역시 미추홀구 경인북길357번길 6 (주안2동 590-22번지 일원)
- **시공사**: 라인건설 (브랜드: 주안 센트럴 파라곤)
- **규모**: 지하 2층 ~ 지상 최고 40층, 12개동, 총 1,321세대 (일반분양 767세대)
- **분양**: 2022.05.23 ~ 05.26 (모집공고 2022000222)
- **평면 7종**: 51A / 51B / 59 / 62A / 62B / 76 / 84

## 로컬 실행

```bash
cd paragonPJ
./start.sh        # macOS/Linux
start.bat         # Windows
```

브라우저에서 http://localhost:8000/주안센트럴파라곤.html 접속.

## 계산기 데이터 기준

- **입주지연보상금**: 분양계약서 지체상금 조항 — `ROUNDUP(원금 × 이율 / 365 × 일수, 0)`, 단계별 이율 9.05% → 10.05% → 11.05% → 12.05% (최대)
- **중도금이자**: 62a 변동금리 기준, 5단계 (A~E), 회차별 × 단계별 적용 일수 합산
- **대출금**: 보금자리론 4.0% (HF 정책 / 2026년 1월 기준), 집단대출 4.5% (1~3년 거치 선택), 일반 주담대 4.2%
